Migrate your contact forms automatically from Pirate Forms to Formidable Forms.

== Description ==
[Pirate Forms](https://wordpress.org/plugins/pirate-forms/ "Pirate Forms") is a basic WordPress contact form plugin, but it is no longer getting new updates. Are you looking for contact forms that will do more heavy lifting and automation? Switch easily and instantly to [Formidable Forms](https://wordpress.org/plugins/formidable/ "Formidable Forms"). We'll automatically import your Pirate forms and email notifications for you. Once migrated, you'll have access to the most advanced form builder for WordPress forms.

== Installation ==
1. Go to your Plugins -> Add New page in your WordPress admin
2. Search for 'Pirate Forms to Formidable'
3. Click the 'Install Now' button
4. Activate the plugin through the 'Plugins' menu
5. If you don't have [Formidable Forms](https://wordpress.org/plugins/formidable/ "Formidable Forms") installed, install it now.
6. Go to the Formidable -> Import/Export page
7. Click the 'Start Import' button to migrate from Pirate Forms to Formidable
